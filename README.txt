Reel Co Rental Agreement Tablet App

Use:
1. Upload this folder to GitHub Pages, Cloudflare Pages, Netlify, or another static web host.
2. Open the hosted index.html URL in Chrome on the Android tablet.
3. Tap Install App or Chrome menu > Add to Home screen.
4. The form autosaves to that tablet and works offline after first load.

PDF:
Tap Submit & Generate PDF, then choose Save as PDF or your printer from Android Chrome.

Backup:
Use Export Backup before replacing tablets or clearing Chrome data. Use Import Backup to restore.
